# RFData - Clínica Financiera DIY
Aplicación de auditoría patrimonial y planificación financiera automática.

## Características
- Auditoría de flujos de caja y fondo de emergencia.
- Simulación de impacto inflacionario.
- Cálculo de capacidad vs voluntad de inversión.
- Generación de informes financieros en PDF.
- Sistema de leads con envío automático por email.

## Despliegue
Esta app está construida con Python y Streamlit.
